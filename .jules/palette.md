## 2025-05-03 - Improving Navbar Accessibility with Semantic Buttons
**Learning:** Converting interactive `div` elements to semantic `button` elements is crucial for keyboard accessibility and screen reader support. Using Bootstrap 5 utility classes like `btn focus-ring focus-ring-primary p-0 border-0` allows for maintaining visual parity while providing necessary focus indicators and accessibility features.
**Action:** Always prefer semantic `<button>` elements for interactive components. Apply `aria-label` to icon-only buttons and ensure focus states are visible using utility classes.

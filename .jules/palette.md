# Palette UX Journal

## 2026-04-25 - Semantic Buttons and Focus Rings
**Learning:** Replacing non-semantic interactive `div` elements with `button` tags provides native keyboard support and screen reader recognition. Using Bootstrap 5's `.focus-ring` utility ensures high-contrast focus indicators without custom CSS.
**Action:** Always prefer `<button>` for triggers and apply `.focus-ring` for consistent accessibility.
